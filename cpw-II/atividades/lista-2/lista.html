Lista de Exercícios 1


1) Faça um programa que crie um array com 50 números aleatórios de 0 a 100 e mostre o maior número.


2) Faça um programa que crie um array com 50 números aleatórios de 0 a 100 e calcule a média dos elementos.


3) Faça um programa que crie um array com 50 números aleatórios de 0 a 100 e mostre os 3 números que mais se repetiram.


4) Faça um programa que crie um array com 50 números aleatórios de 0 a 100 e mostre quantas vezes eles se repetiram.


5) Faça um programa que crie um array com 50 números aleatórios de 0 a 100 e mostre o menor número.
